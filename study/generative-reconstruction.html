# 生成式重建：从几何恢复到内容生成

本文档为《生成式重建》洞察初稿，基于 Feed-Forward 3D Scene Modeling 综述及相关文献整理

1. 引言：重建范式的演变

1.1 从传统重建到神经重建

三维重建是计算机视觉的核心任务之一，其目标是**从二维观测中恢复三维世界的几何与外观**。传统方法如 SfM (Structure-from-Motion) [9]、MVS (Multi-View Stereo) [10] 通过多视图几何约束实现高精度重建，但存在显著局限：

  • **逐场景优化**：每个场景需要独立的优化过程，耗时从分钟到小时不等
  • **类别特定**：难以跨类别泛化，训练数据与测试场景需保持分布一致
  • **稀疏输入敏感**：在视角稀疏或遮挡严重场景下性能急剧下降

神经重建方法（NeRF [1]、3DGS [2]）通过可微渲染实现了照片级真实感的新视角合成，但仍继承了逐场景优化的范式。

1.2 前馈重建的兴起

**前馈重建 (Feed-Forward Reconstruction)** 代表了一种根本性的范式转变：

**核心思想**：学习一个从图像到3D表示的映射函数，通过单次前向传播完成重建，无需测试时优化。

这一范式的优势在于：

  • **高效推理**：单次前向传播即可完成重建
  • **跨场景泛化**：在大规模数据上训练，可泛化到未见场景
  • **端到端集成**：可无缝嵌入下游任务流水线

1.3 生成式重建的定义

**生成式重建 (Generative Reconstruction)** 是前馈重建的自然延伸，其核心特征是：

**利用生成模型的先验知识（来自大规模2D数据或3D数据），在重建过程中"生成"缺失信息，实现从稀疏观测到完整3D场景的恢复。**

与传统重建的"恢复已有信息"不同，生成式重建强调"推断缺失信息"——这使其能够：

  • 从单张图像生成完整3D场景
  • 在遮挡区域"幻觉"出合理的内容
  • 生成符合物理规律和语义一致的重建结果

2. 3D表示方法：生成的载体

生成式重建的输出需要合适的3D表示。不同的表示方法决定了生成的灵活性、质量和下游可用性。

2.1 神经辐射场 (NeRF)

**NeRF** [1] 将场景表示为位置-方向的连续函数：


MLP(x, d) = (c, σ)

其中 x=(x,y,z) 为3D位置，d=(θ,ϕ) 为观察方向，输出颜色 c 和密度 σ。

**前馈 NeRF 方法**：

  • **PixelNeRF** [13]：开创性的条件化辐射场，从图像特征预测 NeRF
  • **IBRNet** [84]：使用 ray transformer 学习连续的视角插值函数
  • **VisionNeRF** [88]：使用 ViT 提取全局上下文特征

**优势**：高质量新视角合成，处理复杂遮挡和视角相关效果

**局限**：训练和推理需要密集的 MLP 查询，内存开销大

2.2 3D高斯泼溅 (3DGS)

**3DGS** [2] 使用各向异性3D高斯基元显式表示场景：


G = {(μⱼ, Σⱼ, αⱼ, cⱼ)}ⱼ=1^N

其中 μ∈ℝ³ 为位置，Σ∈ℝ³ˣ³ 为协方差矩阵，α∈[0,1] 为不透明度，c∈ℝ³ 为颜色。

**前馈 3DGS 方法**：

  • **pixelSplat** [15]：从图像直接预测像素对齐的3D高斯参数
  • **MVSplat** [17]：利用平面扫描代价体定位高斯中心
  • **GS-LRM** [93]：适配 LRM 架构直接输出 3DGS 参数

**优势**：实时渲染，训练快速（分钟级），显式表示便于编辑

**局限**：需要大量高斯基元（百万级），内存占用高

2.3 点图 (Pointmap)

**点图** 将场景表示为密集的2D-3D对应关系：


X ∈ ℝ^H×W×3

**前馈点图方法**：

  • **DUSt3R** [3]：开创性的密集坐标回归方法，同时预测点图和相机位姿
  • **MASt3R** [26]：增加密集局部特征头，实现鲁棒匹配
  • **VGGT** [19]：基础模型，通过多任务预训练学习丰富的视觉-几何特征空间

**优势**：直接建立2D-3D对应，便于视觉定位

**局限**：表示能力有限，难以表达复杂外观

2.4 其他表示

  • **隐式几何表示**：Occupancy Networks [5]、DeepSDF [4] 使用连续函数表示形状
  • **网格 (Mesh)**：直接兼容图形管线，但难以处理拓扑复杂场景
  • **三平面 (Triplane)** [75,76]：紧凑的中间表示，便于2D卷积处理
  • **光场** [14]：参数化为4D函数，实时渲染但难以提取表面

3. 生成式重建的核心技术

3.1 特征增强：从2D到3D的提升

隐式特征图的质量直接决定3D场景的解码质量。

3.1.1 架构演进

**CNN 时代**：

  • PixelNeRF [83]：全卷积条件化，从单/少视图预测 NeRF
  • Convolutional Occupancy Networks [64]：结合卷积编码器和隐式解码器

**Transformer 时代**：

  • **SRT** [87]：引入 transformer 解码器处理图像特征
  • **LRM** [75]：大型重建模型，使用 ViT 编码2D图像，transformer 解码为3D三平面
  • **GRM** [92]、**GS-LRM** [93]：适配 LRM 架构直接输出 3DGS

**状态空间模型**：

  • **Gamba** [97]：基于 Mamba 的序列预测，线性复杂度
  • **Long-LRM** [33]：处理长序列（~250k tokens，32视图）

3.1.2 跨视图融合

多视图信息融合是生成一致3D场景的关键。

**特征级融合**：

  • **AttnRend** [99]：多视图 ViT 编码器，跨视图注意力
  • **eFreeSplat** [105]：自监督 ViT + 高斯对齐模块
  • **iLRM** [106]：迭代跨视图细化，分解多视图交互为两阶段注意力

**几何级融合**：

  • **DUSt3R** [3]：密集坐标回归，同时预测位姿
  • **MV-DUSt3R** [100]：多视图解码器块，促进跨视图信息交换
  • **MUSt3R** [111]：对称架构，内存机制支持多视图

**长序列处理**：

  • **PreF3R** [102]：空间记忆网络，增量融合时序信息
  • **Spann3R** [101]：外部3D空间记忆，存储点级特征
  • **CUT3R** [107]：压缩状态表示，支持动态场景

3.1.3 视觉基础模型集成

利用大规模预训练模型注入强大的视觉和几何先验。

**视觉特征**：

  • **DUSt3R** [3]：使用 CroCo [213] 预训练权重
  • **Feat2GS** [125]：探测预训练2D模型（分割、扩散、识别），无需重训3D网络
  • **Mono3R** [124]：集成单视图先验，细化多视图立体特征

**文本-视觉融合**：

  • **CATSplat** [94]：利用视觉-语言模型的文本指导增强单图3D重建

3.2 几何感知：解决深度模糊

前馈3D重建的核心挑战是从2D图像鲁棒、准确地推断3D几何。

3.2.1 显式几何聚合

**代价体构建**：

  • **MVSNeRF** [126]：平面扫描变换构建代价体，3D卷积聚合
  • **GeoNeRF** [128]：级联代价体 + Transformer 渲染器
  • **MuRF** [129]：目标视图锥体体积，有效聚合跨视图特征

**对应关系推理**：

  • **SRF** [127]：神经立体框架，避免完整代价体构建
  • **GPNR** [79]：沿极线提取局部斑块，Transformer 聚合
  • **MatchNeRF** [134]：特征余弦相似度作为几何先验

**表面感知表示**：

  • **SparseNeuS** [65]：SDF 表面 + 体渲染
  • **VolRecon** [66]：SRDF + 基于射线的渲染
  • **SurfaceSplat** [70]：SDF 粗几何 + 3DGS 细化细节

3.2.2 后细化

迭代改进生成的基元以捕获复杂几何。

  • **HiSplat** [137]：层次化方法，粗高斯 + 细高斯 + 误差感知模块
  • **FreeSplat** [138]：像素级对齐，逐步聚合更新局部和全局高斯三元组
  • **PixelGaussian** [139]：动态框架，根据局部几何复杂度自适应调整高斯分布
  • **G3R** [140]：结合快速前馈预测和基于梯度的细化

3.2.3 无位姿重建

从未知相机参数的图像中重建，同时推断几何和位姿。

**点图路线**：

  • **DUSt3R** [3]：成对重建 = 密集点图回归
  • **π³** [153]：全排列等变架构，预测仿射不变位姿和尺度不变点图

**3DGS 路线**：

  • **Splatt3R** [144]：适配 MASt3R 框架预测高斯属性
  • **NoPoSplat** [145]：在单个规范输入视图的局部坐标系中预测所有高斯
  • **PF3plat** [146]：预训练单目深度 + 对应模型粗对齐，轻量模块细化
  • **AnySplat** [152]：几何 transformer + 可微位姿估计模块

3.2.4 预训练几何指导

直接注入来自强大预训练模型的几何线索。

**深度先验**：

  • **DepthSplat** [18]：多视图深度估计 + 3DGS 的协同链接
  • **Flash3D** [156]：单目深度估计扩展为完整3D重建器
  • **Niagara** [157]：集成单目深度和法线估计

**点图先验**：

  • **PM-Loss** [28]：基于前馈点图模型的正则化损失，强制几何平滑

3.3 生成增强：利用生成先验

这是"生成式重建"的核心——利用生成模型（尤其是扩散模型）的先验知识增强重建。

3.3.1 数据增强

扩充训练分布，提升泛化能力。

  • **Puzzles** [174]：从单张图像/视频合成无限位姿视频-深度数据
  • **MegaSynth** [175]：程序化生成数十万非语义3D场景
  • **Aug3D** [176]：基于 SfM 的新视角增强户外数据集
  • **MVBoost** [177]：多视图生成模型 + 重建一致性检查生成伪真值

3.3.2 视觉增强

利用生成先验增强渲染输出，抑制伪影，幻觉合理细节。

**扩散先验集成**：

  • **MVSplat360** [178]：3DGS + Stable Video Diffusion，高斯渲染特征注入扩散潜空间
  • **LatentSplat** [180]：变分3D高斯，显式编码潜空间不确定性
  • **ProSplat** [179]：单步扩散细化阶段增强高斯渲染视图
  • **DIFIX3D+** [181]：单步扩散模型 Difix 作为神经增强器，抑制残留伪影

**混合策略**：

  • **CogNVS** [264]：共视像素3D重建 + 隐藏像素视频扩散修复，自监督扩散模型测试时微调

3.4 时序感知：动态4D重建

将生成式重建扩展到动态场景。

3.4.1 在线流式处理

逐帧处理，增量更新场景表示。

  • **StreamSplat** [182]：估计每高斯速度和小变形，单次传播更新运动和外观
  • **DGS-LRM** [183]：Plücker 射线查询 + 时空 Transformer，从单目视频预测可变形高斯
  • **CUT3R** [107]：持久循环状态跨帧携带几何
  • **Stream3R** [184]：decoder-only 因果 Transformer，可扩展序列3D重建

3.4.2 离线处理

聚合完整片段，单次前向传播预测全局一致的4D表示。

  • **L4GM** [186]：LRM 扩展到时间维度，单目视频 → 每帧高斯
  • **4D-LRM** [190]：联合建模空间和时间，输出4D高斯场
  • **MonST3R** [187]：DUSt3R 微调用于动态视频
  • **4DGT** [192]：滚动时间窗口直接预测一致的4D高斯

3.4.3 交互建模

允许用户注入力、编辑内容或调整材质，实时反馈物理合理结果。

  • **PIXIE** [197]：离线重建几何 + 前馈预测材质场 → 物理求解器动画
  • **PhysGM** [198]：单次前向传播同时预测3D高斯场景和物理属性

4. 模型效率：走向实时部署

4.1 特征效率

学习"在哪里"和"如何"聚合多视图信息，最小化不必要的计算。

  • **ENeRF** [161]：学习深度引导采样器，显著减少每射线查询
  • **ProNeRF** [162]：投影感知采样，动态调整不透明度
  • **TinySplat** [163]：免训练感知和空间压缩
  • **ZPressor** [164]：IB 引导锚点-支持交叉注意力，融合为紧凑潜变量 Z

**VGGT 系列优化**：

  • **FastVGGT** [166]：免训练 token 合并
  • **QuantVGGT** [167]：训练后量化
  • **Sparse VGGT** [168]：自适应块稀疏核
  • **Evict3R** [169]：免训练 KV 缓存驱逐策略
  • **LiteVGGT** [171]：几何感知缓存 token 合并
  • **Speed3R** [170]：稀疏双分支设计

4.2 表示压缩

显式高斯压缩。

  • **GGN** [141]：高斯图上的消息传递，池化合并和剪枝
  • **PixelGaussian** [139]：级联适配器引导剪枝和分裂
  • **FreeSplat++** [172]：像素级三元组融合去重，加权浮点去除
  • **LongSplat** [173]：GIR 空间身份感知冗余压缩

5. 数据集与基准

5.1 数据集分类

**几何导向数据集**：提供可靠的3D表示真值（点云、深度、位姿）

  • DTU [21]、ScanNet [22]、Replica [23]、Waymo [281]

**视觉导向数据集**：来自野外或策划视频，适合新视角合成

  • NeRF-Synthetic [1]、RealEstate10K [24]、DL3DV [25]

5.2 评估指标

**新视角合成**：

  • PSNR（峰值信噪比）
  • SSIM（结构相似性）
  • LPIPS（学习感知图像块相似度）

**相机位姿**：

  • RTA（相对平移精度）
  • RRA（相对旋转精度）
  • AUC（曲线下面积）

**点图**：

  • Accuracy（精度）：预测点到真值表面的平均距离
  • Completeness（完整性）：真值点到预测重建的平均距离
  • Chamfer Distance：精度和完整性的综合

6. 应用场景

6.1 自动驾驶

大规模动态重建，严格的实时和时序一致性要求。

  • **Driv3R**、**Omni-Scene**：处理大规模户外场景
  • **LoRA3D**：参数高效微调适应驾驶场景

6.2 机器人

快速可泛化的3D表示支持下游决策。

  • **操作**：精确的物体几何重建
  • **导航**：显式场景记忆支持长视野规划

6.3 场景理解

前馈3D先验作为语义场景理解的骨干。

  • **Uni3R** [117]：联合重建语义3D高斯基元，统一渲染、深度预测、开放词汇3D理解

6.4 SfM 和 SLAM

  • **VGGT-SLAM**：基于 VGGT 的视觉定位和建图
  • **LoGeR** [121]：分块处理长视频，结合测试时训练记忆和滑动窗口注意力

6.5 视频生成

几何感知的视频生成。

  • **4Real-Video-V2** [193]：VGGT 基础上引入高斯头和动态层
  • **MoVieS** [194]：共享编码器 + 三个独立头（外观、深度、运动）

9. 结语

生成式重建代表了3D视觉从"恢复"到"生成"的范式转变。通过整合生成模型的强大先验，我们能够在稀疏观测下推断完整的3D场景，实现从"看到什么重建什么"到"看到什么推断什么"的跨越。

这一领域的快速发展得益于：

1. **大规模预训练模型**：提供丰富的2D视觉和几何先验

2. **高效3D表示**：3DGS 等显式表示支持实时渲染和编辑

3. **前馈架构**：单次传播实现高效推理和跨场景泛化
